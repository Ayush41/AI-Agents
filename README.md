# AI-Agents

This repository is dedicated to learning about **frameworks of AI agents** and the **concepts** I am exploring. Here, you'll find implementations, examples, and projects built with various AI agent frameworks to understand their capabilities, use cases, and best practices. This is a personal learning journey through different AI agent frameworks and the fundamental concepts behind them.

## Overview

This repository focuses on learning about **AI agent frameworks** and the **core concepts** that power them. AI agents are autonomous systems that can perceive their environment, make decisions, and take actions to achieve specific goals. Through hands-on projects and examples, I'm exploring popular AI agent frameworks to understand their architectures, design patterns, and practical applications.

## Framework Directories

Explore the following frameworks to learn about different approaches to building AI agents:

### 🤖 [Langchain Agents](./Langchain-Agents/)
Learn about building AI agents using LangChain, a framework for developing applications powered by language models. LangChain provides tools for creating chains of reasoning and agent-based applications.

### 👥 [CrewAI Agents](./CrewAI-Agents/)
Explore CrewAI, a framework for orchestrating role-playing, autonomous AI agents. CrewAI enables you to build multi-agent systems where agents collaborate to solve complex tasks.

### 📊 [LangGraph](./LangGraph/)
(Coming Soon) Learn about LangGraph, a library for building stateful, multi-actor applications with LLMs. LangGraph extends LangChain with graph-based workflows.

### 🛠️ [ADK (Agent Development Kit)](./adk/)
(Coming Soon) Explore ADK for building and deploying AI agents with advanced capabilities.

### 🔄 [AutoGen](./AutoGen/)
(Coming Soon) Learn about Microsoft AutoGen, a framework that enables the development of LLM applications using multiple agents that can converse with each other.

### 🔗 [N8N Agents](./N8N-Agents/)
(Coming Soon) Explore building AI agents using N8N, a workflow automation tool that can integrate AI capabilities.

### 🌊 [Langflow](./Langflow/)
(Coming Soon) Learn about Langflow, a visual programming environment for building AI applications with LangChain.
